# Melato AI Concierge Roadmap

## Phase 1 - Built now, $0

- Static website chat widget
- Brand-trained response engine
- Customer-service intent routing
- Email reply draft generator
- Browser voice mode
- PWA install support
- GitHub Pages deploy workflow
- Android debug APK workflow through Capacitor

## Phase 2 - Still $0 if self-hosted

- Editable local knowledge base JSON
- Product CSV import for names, links, fabric, fit, and care
- Conversation export for manual support review
- Optional local LLM adapter through Ollama
- Internal-only admin mode

## Phase 3 - Paid or infrastructure-dependent

- True inbound phone number agent
- SMS support
- Live Shopify order lookup
- Gmail auto-draft or auto-reply
- Helpdesk ticket creation
- Customer authentication

## Hard truth

A real phone-call agent that answers Melato's phone number needs a telephony provider such as Twilio, Telnyx, Vonage, or a Shopify/helpdesk phone integration. That is not truly $0. The current free alternative is browser voice mode.
